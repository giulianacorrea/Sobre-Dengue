# 🦟Sobre 

Este projeto aborda informações essenciais sobre a dengue, reunindo conteúdos educativos e preventivos para conscientizar o público sobre a doença. Desenvolvido como atividade da disciplina de Programação Web do curso técnico em Informática, o site apresenta dados claros, acessíveis e organizados sobre sintomas, prevenção e cuidados.
Acesse: https://sobredengue.vercel.app/
